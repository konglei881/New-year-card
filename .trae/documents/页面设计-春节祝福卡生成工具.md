# 页面设计说明（桌面端优先）

## 全局样式（Design Tokens）
- 背景色：`#FFF7E6`（暖米色）
- 主色：`#D32F2F`（春节红）
- 辅色：`#C9A227`（金色点缀）
- 文本色：`#1F1F1F`；次级文本：`#6B7280`
- 字体：系统字体优先；标题加粗（600–700）；正文 14–16px；标题 24–32px
- 按钮：主按钮红底白字；hover 加深 6%；disabled 降低不透明度并禁用点击
- 卡片：白底、圆角 12px、阴影轻量；模块间距 16–24px
- 交互反馈：校验错误使用红色提示与输入框描边；生成/下载提供 loading 状态

---

## 页面 1：祝福卡生成页（首页）
### Layout
- 桌面端三栏混合布局（CSS Grid）：左侧“表单”，中间“预览”，右侧“模板/风格”。
- 栅格：`grid-template-columns: 360px 1fr 320px`；列间距 24px；内容区最大宽 1200px 居中。
- 响应式：
  - ≤1024px：表单与模板折叠为上下两块，预览居中（两列或单列）。

### Meta Information
- Title：春节个人专属祝福卡生成
- Description：填写称呼与署名，一键生成可下载春节祝福卡。
- Open Graph：标题同上；预览图为默认模板缩略图。

### Page Structure
1) 顶部导航（Header）
2) 主内容区（三栏）
3) 底部说明（Footer：合规提示/版权说明）

### Sections & Components
- Header
  - 左：产品名
  - 右：帮助入口（文本链接，弹出说明对话框即可）
- 左栏：信息表单（Card）
  - 字段：收祝福者称呼/关系（必填）、你的署名（必填）、自定义祝福语（可选）
  - 校验展示：字段下方错误文案；首个错误自动聚焦
  - 主按钮：一键生成（校验通过才可用）
- 中栏：实时预览（Card）
  - 预览画布容器：保持固定比例（如 9:16），显示安全边距参考线（仅预览态）
  - 状态：空态（提示先填写/选择模板）、生成中（loading）、生成完成（可点击放大）
- 右栏：模板与风格（Card）
  - 模板列表：网格缩略图（2 列），支持选中态描边
  - 风格选项：配色/装饰强度（低/中/高）
- Footer
  - 合规提示：不支持不当内容；下载为本地生成，不默认保存

## 页面 2：预览与下载页
### Layout
- 双栏布局（Flexbox）：左侧高清预览，右侧下载与说明。
- 左侧预览区占比 65%，右侧控制面板 35%，最大宽 1200px。

### Meta Information
- Title：预览与下载｜春节祝福卡
- Description：确认效果并下载 PNG/JPG。
- Open Graph：使用本次生成预览图（若可获取）。

### Page Structure
1) 顶部操作条
2) 预览 + 控制面板
3) 提示与验收点

### Sections & Components
- 顶部操作条（Sticky）
  - 左：返回编辑（按钮）
  - 右：再次生成（次按钮，可选）
- 左侧：高清预览区
  - 支持 1:1 显示与“适配窗口”切换（按钮/开关）
- 右侧：下载面板（Card）
  - 选择：格式（PNG/JPG）、尺寸（1080×1920/1200×1200）
  - 下载按钮：下载到本地（loading/disabled 状态明确）
  - 生成说明：展示本次参数摘要（模板名、风格、是否自定义祝福语）
- 错误/兜底弹层
  - 生成失败：提示重试、切换模板、减少装饰强度
  - 下载失败：提示改用另一格式或刷新页面

### Interaction States
- 下载成功：展示轻提示（不阻断），并保持可再次下载。
- 返回编辑：保留当前参数并回填到首页表单与选项。
